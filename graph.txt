Graph Example (Adjacency List Format)

A : B C
B : D E
C : F

Explanation:
- Node A connects to B and C
- Node B connects to D and E
- Node C connects to F
- Node D has no neighbors
- Node E connects to F
- Node F has no neighbors

This graph will be used for BFS, DFS, and shortest path demonstrations.
